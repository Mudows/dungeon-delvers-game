### Visão Geral do Projeto
Dungeon Delvers é um roguelike top-down para PC, com jogabilidade baseada em turnos em grid, pixel art 16×16 no estilo Dark Fantasy, desenvolvido na engine Construct 3. O projeto tem como objetivo entregar uma experiência de progressão, tomada de decisão e rejogabilidade, com escopo inicial definido por um MVP de 3 andares temáticos.

**Arquitetura Atual**

O projeto está estruturado em módulos JS independentes, cada um com responsabilidade única:

- `grid.js` — conversão de coordenadas e movimentação com detecção de colisão e _bump attack_.
- `map.js` — geração procedural via BSP com MST para conectividade garantida.
- `themes.js` — configuração visual de tiles por fase, com sorteio ponderado.
- `turn.js` — ciclo de turnos, factory de inimigos e classe base Enemy.
- `combat.js` — fórmulas de dano físico conforme especificação MSAT.
- `hud.js` — interface do jogador integrada ao editor do C3.
- `enemies.json` — dados externos de famílias de inimigos (stats, sprite, chance de spawn).
- `main.js` — orquestração geral e input.
